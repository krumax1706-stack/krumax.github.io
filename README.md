# 🐘 สื่อการเรียนการสอนภาษาไทย ชั้นประถมศึกษาปีที่ 3 (Thai Learning P3)

เว็บไซต์นี้เป็นสื่อการเรียนรู้แบบตอบโต้ (Interactive Web Application) ที่พัฒนาขึ้นสำหรับนักเรียนชั้นประถมศึกษาปีที่ 3 เพื่อช่วยฝึกฝนทักษะพื้นฐานหลักภาษาไทย 3 ระดับ พร้อมแบบทดสอบวัดผลการเรียนรู้ตามอนุกรมวิธานของบลูม (Bloom's Taxonomy)

---

## 🌟 คุณสมบัติเด่นของระบบ (Key Features)
1. **การออกแบบเชิงประสบการณ์ผู้ใช้ (Premium Design):** ใช้โทนสีพาสเทลที่เป็นมิตร ลายเส้นสะอาด สบายตา เหมาะสำหรับเด็กช่วงอายุ 8-9 ปี พร้อมมีมาสคอต "พี่ปันดี (P'PanDee)" นำทางการเรียนรู้
2. **แอนิเมชันตอบสนอง (Interactive Animations):** แสดงการ์ดสะบัด มีกล่องคำพูดข้อความอธิบาย มี Confetti ฉลองเมื่อทำข้อสอบถูกต้อง และระบบสั่นเมื่อตอบผิดเพื่อกระตุ้นการวิเคราะห์ใหม่
3. **กิจกรรมฝึกฝนปฏิบัติจริง (Creative Playgrounds):** มีห้องทบทวนเนื้อหาก่อนทำข้อสอบจริง:
   - *เขียนสะกดคำศัพท์* (ป้อนข้อมูลและตรวจผลทันที)
   - *บล็อกเรียงประโยค* (คลิกสลับบล็อกประธาน-กริยา-กรรม)
   - *แต่งเรื่องสั้นจากภาพและคีย์เวิร์ด* (มีระบบเช็คลิสต์ตรวจเงื่อนไขด้วยตนเอง)
4. **แบบประเมินอ้างอิงตาม Bloom's Taxonomy:** มีรายงานวิเคราะห์ทักษะสมองของนักเรียนทั้ง 6 ระดับ (จำ, เข้าใจ, ประยุกต์ใช้, วิเคราะห์, ประเมินค่า, สร้างสรรค์) ช่วยให้คุณครูหรือผู้ปกครองเห็นจุดแข็งและส่วนแก้ไขได้ทันที
5. **เกียรติบัตรอิเล็กทรอนิกส์ (Downloadable Certificate):** หากเรียนจบและผ่านเกณฑ์ สามารถกรอกชื่อเพื่อสร้างและดาวน์โหลดเกียรติบัตรเป็นภาพ `.png` นำไปอวดครอบครัวหรือเก็บสะสมได้เลย!

## 📂 โครงสร้างโฟลเดอร์ (Folder Structure)
```
thai-learning-p3/
├── index.html               # หน้าจอหลัก แดชบอร์ด และหน้าต่างประเมิน (SPA)

│   ├── styles.css           # โครงร่างสี สไตล์หลัก และ Layout Responsive
│   ├── animations.css       # เอฟเฟกต์ Confetti, เด้ง, สั่น, และเลื่อนเฟด
│   └── components.css       # สไตล์การ์ด, บล็อกตัวหนังสือ, โมดอล และปุ่ม

│   ├── content.js           # ฐานข้อมูลบทเรียนและแบบทดสอบ Bloom's Taxonomy

    └── images/
        └── mascot.png       # รูปภาพพี่ปันดีนำทาง (AI Generated)
```

## 🚀 วิธีการนำไป Deploy บน GitHub Pages (Deployment Guide)
เนื่องจากระบบนี้เขียนด้วย **HTML, CSS, และ JavaScript แบบดั้งเดิม (Vanilla Web)** โดยไม่มีการใช้ Framework ที่ซับซ้อน จึงทำให้โหลดเร็วมากและสามารถเปิดใช้งานฟรีผ่าน GitHub Pages ได้ใน 4 ขั้นตอน:

1. **สร้าง Repository บน GitHub:**
   - เข้าเว็บไซต์ [GitHub](https://github.com/) และเข้าสู่ระบบ
   - กดปุ่ม **New** เพื่อสร้าง Repository ใหม่ ตั้งชื่อเช่น `thai-learning-p3`
   - เลือกเป็น **Public** และกดยืนยันสร้างคลังโค้ด

2. **อัปโหลดไฟล์ขึ้น GitHub:**
   - **กรณีใช้โปรแกรม Git (Command Line):**
     เปิด Git Bash / Terminal ในโฟลเดอร์โครงการนี้ แล้วรันคำสั่ง:
     ```bash
     git init
     git add .
     git commit -m "Initial commit for Thai Learning Media"
     git branch -M main
     git remote add origin https://github.com/USERNAME/thai-learning-p3.git  # เปลี่ยน USERNAME เป็นชื่อไอดี GitHub ของคุณ
     git push -u origin main
     ```
   - **กรณีอัปโหลดผ่านเว็บเบราว์เซอร์:**
     ไปที่หน้า GitHub Repository ที่เพิ่งสร้างใหม่ กดปุ่ม **"uploading an existing file"** แล้วลากไฟล์ทั้งหมดในโฟลเดอร์ `thai-learning-p3` (รวมถึงโฟลเดอร์ย่อย `css`, `js`, `assets`) ไปวางในเว็บเบราว์เซอร์ แล้วกด **Commit changes**

3. **เปิดใช้งาน GitHub Pages:**
   - เข้าไปที่หัวข้อ **Settings** ด้านบนของ GitHub Repository ของคุณ
   - เมนูด้านซ้าย เลือก **Pages**
   - ที่หัวข้อ **Build and deployment** -> **Source** ให้ตั้งค่าเป็น `Deploy from a branch`
   - ในส่วนของ **Branch** ให้เปลี่ยนจาก `None` เป็น `main` (หรือ `master`) และเลือกโฟลเดอร์เป็น `/ (root)` จากนั้นกด **Save**

4. **เข้าชมเว็บไซต์:**
   - รอระบบ GitHub ประมวลผลประมาณ 1-2 นาที
   - รีเฟรชหน้าเว็บหน้าเดิม คุณจะพบลิงก์ทางเข้าเว็บอยู่ด้านบน เช่น `https://USERNAME.github.io/thai-learning-p3/`
   - คุณสามารถนำลิงก์นี้ส่งให้เด็ก ๆ เข้ามาศึกษาเรียนรู้ได้ทันทีทุกที่ทุกเวลา!
